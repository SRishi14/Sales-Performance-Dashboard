# Sales-Performance-Dashboard

A Power BI dashboard built on 500K rows of sales data.

## What's inside
- `SalesDashboard.pbix` — Power BI report file
- `sales_data.csv` — Raw sales dataset

## Dashboard pages
- Revenue overview
- Sales by Region & Country
- Product & Item Type analysis
- Sales Channel performance
- Order Priority & Shipping operations

## Tools used
- Power BI Desktop
- DAX for measures
- Power Query for data cleaning
